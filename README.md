Superstore Sales Data Analysis
# 📊 Project Overview

This project focuses on analyzing retail sales data from a Superstore dataset using Python. The goal is to explore sales patterns, understand category performance, and generate meaningful business insights through data analysis and visualization.

The project demonstrates the complete data analysis workflow, including data cleaning, feature engineering, exploratory data analysis (EDA), and interactive visualizations.

## 🎯 Project Objectives

Analyze sales trends over time

Identify top-performing product categories

Understand customer purchasing patterns

Create interactive visualizations for better insights

Demonstrate practical data analysis skills using Python

### 🛠 Tools & Technologies Used

Python

Pandas – Data cleaning and manipulation

Plotly – Interactive data visualization

Jupyter Notebook – Analysis and documentation

##### 📂 Dataset

The dataset used in this project is the Superstore Sales Dataset, which contains information such as:

Order Date

Ship Date

Sales

Category

Customer Details

Product Information

Region and Geography

######🔎 Data Analysis Steps
1️⃣ Data Loading

The dataset was loaded using Pandas and inspected to understand the structure.

2️⃣ Data Cleaning

Checked data types

Handled formatting issues

Converted Order Date and Ship Date into datetime format

3️⃣ Feature Engineering

Additional time-based features were created:

Order Month

Order Year

Day of Week

These features helped analyze seasonal sales trends.

4️⃣ Exploratory Data Analysis (EDA)

Key analysis performed:

Monthly Sales Trend

Sales by Product Category

Sales Distribution across categories

Time-based sales patterns

######📈 Visualizations

Interactive charts were created using Plotly, including:

 Monthly Sales Trend Line Chart

 Sales by Category Pie Chart

 Additional charts to explore sales patterns

These visualizations help identify business trends and performance insights.
